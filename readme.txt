== Description ==
Image Composer is an official plugin maintained by the Prozoned team that make  two  seprate or diffrent images combined in one single images.
It provides some nice features , and will be fully supported and maintained until at least 2022,or as long as is necessary.
 
 = Plugin Features =


*   It will allow you to add image over image.
*   Simplest easy ever to use.
  

== Installation ==
This section describes how to install the plugin and get it working.

    Option 1:
1. Go to WordPress Dashboard. Locate Plugins -> Add New
2. Search image-composer plugin using search option
3. Find the plugin and click Install Now button
4. After installation, click on Activate Plugin link to activate the plugin.
     or
   Option 2:

1. Download the plugin image-composer.zip
2. Go to WordPress Dashboard. Locate Plugins -> Add New
3. Click on Upload Plugin link from top
4. Upload the downloaded image-composer.zip file and click on Install Now
5. After installation, click on Activate Plugin link to activate the plugin.

== Frequently Asked Questions ==
= Does this plugin work with newest WP version and also older versions? =
Yes, this plugin works really fine with WordPress 5.2!
It  also works with WP below version - but we only tested extensively with WP 5.2.
